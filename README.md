
  # 3D Movie Data Visualization

  This is a code bundle for 3D Movie Data Visualization. The original project is available at https://www.figma.com/design/BVH4YGwFvufpZZUZaThHbH/3D-Movie-Data-Visualization.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to GitHub Pages

  This project must be deployed from the built `dist` output, not from the repository root.

  A GitHub Actions workflow is included at `.github/workflows/deploy-pages.yml` and will automatically build and publish the app to GitHub Pages when you push to the `main` branch.
